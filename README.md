☕ Coffee Shop Management System

A full-stack web application for managing a coffee shop.  
The project allows users to browse products, place orders, and manage coffee shop operations through a modern and responsive interface.

🚀 Project Overview

This application was developed as a practical full-stack project combining a modern frontend with a robust backend architecture.

## The system includes:

- Product management
- Order management
- REST API communication
- Responsive user interface
- Backend and frontend integration
- Database interaction
- Secure API configuration

 🛠️ Technologies Used

## Frontend
Built with modern web technologies:

- React.js
- JavaScript
- HTML5
- CSS3
- Axios
- React Hooks

### Frontend Features
- Dynamic user interface
- Product display
- Order creation
- API consumption
- Responsive design

## Backend
Developed using Spring Boot ecosystem:

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Spring Security
- Maven

### Backend Features
- RESTful APIs
- CRUD operations
- Order processing
- Product management
- CORS configuration
- Security configuration
